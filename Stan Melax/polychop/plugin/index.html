Polygon Reduction Plugin for 3DS Max

Discussions, papers, and pictures are nice, but they dont deliver technology in its most useful form. For those who are interested in trying out polygon reduction, we have provided a polygon reduction modifier plugin for Kinetix 3D Studio Max. The software available here is not a commercially supported product. However, I have already received lots of good feedback on this version.  However, 3ds max now includes a good mesh simplification tool called multi-res.  I’ve kept this web page mostly for reference reasons.

To get the plugin, download:

polychop42.zip 3DS Max 4.2 (special thanks to David Wulff for updating the code, project .dsp file, and compiling the new plugin file polychop42.dlm)

pcp_code.zip Source code - if you ever want to experiment with your own mesh simplification metrics or add other features. (be sure to grab the new changes out of the 4.2 version too)

Unzip the file and copy POLYCHOP.DLM into 3dsmax's "plugins" subfolder. The next time you invoke Max, a modifier named PolyChop should be available.

Release Notes

As mentioned, this version isn't perfect. Some of the things to note:

  • This software comes with no gaurantees or obligations for support.
  • This version doesn't work with some types of multi piece objects.
  • Problems with animating objects.
  • There are numerous techniques for reducing polygonal models. We have only had the time to implement 1 simple method.
  • There is no progress bar. Note that max may pause (without user feedback) for a few seconds while huge models are being reduced for the first time.
  • Source hasn't changed much over the years.
  • There are some features such as "freezing" selected vertices to prevent them from being "collapsed". i.e. with the right option specified, selected vertices should be the last to be decimated. I also added support for maintaining UVW2 info, as well as the option to try to use it to influence the reduction - although the results may not be what you want depending on how you are using the uvw2 channel.
